# 🛒 E-commerce Vanilla JS App

A client-side e-commerce application built with Vanilla JavaScript featuring product listing, cart management, and dynamic total price calculation using local JSON data.

---

## 🚀 Features

* 📦 Product listing from local JSON file
* 🛒 Add to cart functionality
* ➕ Increase / decrease product quantity
* 💰 Dynamic total price calculation
* 🧠 Client-side state management using JavaScript
* ⚡ Fast and lightweight (no frameworks)

---

## 🧑‍💻 Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript (ES6)  
- Vite (for fast development and build)

---

## 📁 Project Structure

```
├── index.html
├── style.css
├── script.js
├── products.json
└── assets/
```

---

## ⚙️ How It Works

* Product data is stored in a local `products.json` file
* JavaScript fetches and displays products dynamically
* Cart functionality is handled using client-side logic
* Total price updates automatically based on cart items
